# SPIRAL ESCAPE

## NOTICE
The task is to develop a hyper-casual game using Unity3D/Contruct or any alternative engine.

## Functions
> 1. Main Menu
   - Play button
   - Settings button
   - Exit button
   - Shop button
   - Reset(score) button
   - User name
   - User avatar
   - BestScore count
>
> 2. Shop menu
   - Coming soon
>
> 3. Settings Menu
   - Show FPS
   - Show Pause/Resume button
   - Home button
>
> 4. GameMenu
   - Pause button
   - Resume button
   - FPS count
   - Score count
>
> 5. GameMenu
   - Restart button
   - Home button
   - Score count
   - User name
   - User avatar
